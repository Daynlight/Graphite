# Security Policy

## Supported Versions

At this time, only the **latest released version** of Graphite receives security updates and patches.

| Version | Supported |
| ------- | --------- |
| Latest  | ✔️        |
| Older   | ❌         |

## Reporting a Vulnerability

If you believe you’ve discovered a security vulnerability, please report it **privately**.

### Preferred Ways to Report

1. **Send an email** to:
   **[danielstodulski000@gmail.com](mailto:danielstodulski000@gmail.com)**

2. **Create a GitHub issue** marked with the `security` label.

   * Do *not* include exploit details publicly.
   * Mention that the full report has been sent by email.

We will review your report as quickly as possible and follow up with next steps.
